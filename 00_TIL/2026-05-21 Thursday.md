###### 문제 12 주식가격(개선 코드를 책 보고 풀음)
- [문제 코드 ](https://school.programmers.co.kr/learn/courses/30/lessons/42584)
- 책 해설 정리
	- 요약: 인덱스를 스택에 넣고 이용하여 stackTop 과 currentIndex의 차이를 이용하여 길이를 구한다
	- 문제 분석하고 풀기
		- 현재 주식 보다 이전 주식의 가격이 높으면 이전 주식의 길이를 확정한다.
		- 길이를 확정한 주식은 이후 계산에서 제외한다.
		- 최근 주식부터 그 이전으로 거슬러 올라가며 비교하므로 스택을 사용하면 된다.
		- 더이상 고려하지 않을 주식은 pop 하면 된다.
		- 스택에 최종적으로 남는 원소들은 길이를 확정하지 않은, 다시 말해 가격이 끝까지 떨어지지 않은 주식일 것이다.
	- 과정
		1. 스택이 비어있으면 비교 대상이 없으므로 인덱스를 스택에 넣는다. 이 때 인덱스를 스택에 넣는 이유는 최종적으로 구해야 할 값은 가격이 떨어지지 않은 기간이기 때문이다.
		2. prices[top] 의 가격과 현재의 주식 가격을 비교하고 현재 가격이 더 크면 스택에 넣는다. 이 때 스택에 push한다는 의미는 길이가 확정되지 않았다는 뜻이다.
		3. 만약 prices[top] 의 가격이 더 크면 주식이 떨어졌다는 뜻이므로 현재의 인덱스와 스택 탑의 인덱스의 차를 이용하여 길이를 확정하고 stack top 의 길이가 확정되었으므로 stack 에서 pop을 한다. 
		4. 이것을 prices 의 길이만큼 반복한다.
		5. stack 에 끝까지 남아있는 것들은 맨 마지막 인덱스의 차로 stack top 과 차를 이용해 길이를 확정하고 pop 해준다.
		6. 이것을 stack 이 빌 때까지 반복한다.
	- 책을 보고 푼 개선 코드
		``` java
		import java.util.*;

class Solution {
    public int[] solution(int[] prices) {
        // 결과를 담을 배열을 만든다.
        int length = prices.length;
        int []result = new int[length];

        // 길이를 재기 위해 인덱스를 push 할 stack 을 만든다.
        Stack<Integer> indexStack = new Stack<>();
        
        // 들어온 배열 끝까지 스택에 넣고 비교해야 한다.(N 반복)
        for(int i = 0; i< length; i++){
            int currentIndex = i;
            
            // stack top 과 현재 넣으려는 값을 비교한다.
            // stackTop이 크면 가격이 떨어졌다는 뜻이므로 
            // 길이를 currentIndex - topIndex 로 지정하여 result 값에 넣어준다.
            //또한, stackTop 의 길이가 정해졌으므로 stackPop 을 해준다.
            //그 아래 stack 도 조사한다.
            while (true){
                // 스택이 비어있으면 비교 자체가 불가하므로 일단 스택에 넣어준다.
                if(indexStack.isEmpty()) {
                    indexStack.push(currentIndex);
                }
                int topIndex = indexStack.peek();
                if(prices[topIndex] > prices[currentIndex]){
                    result[topIndex] = currentIndex - topIndex;
                    indexStack.pop();
                }else{
                    // 가격이 떨어지지 않았으면 다시 stack 에 현재 인덱스를 push 한다.
                    indexStack.push(currentIndex);
                    break;
                }
            }
                
        }
        
        // 끝까지 스택에 남은 것들은 끝까지 가격이 떨어지지 않은 것이므로 맨 마지막 인덱스를 빼준다.
        while(!indexStack.isEmpty()){
            int currentIndex = indexStack.pop();
            result[currentIndex] = length - 1 - currentIndex;
        }
        
        return result;
    }
}
		```

###### 문제 13 크레인 인형 뽑기 게임

- 의사 코드
		1.board 마다 stack 을 만들면 공간 복잡도가 너무 늘어난다. 따라서 board 맨 위의 위치와 수를 나타내는 클래스 배열을 N길이만큼 만든다. 
		![[Pasted image 20260521205938.jpg]]
	2. move 에서 현재 수가 있으면 해당 tops 의 위치로 가서 위치를 +1 한 다음 수를 가져와 바구니에 넣고 길이를 늘린다.
	3. 단, 해당 위치를 늘릴 떄 N 이 되면 -1 로 위치를 바꾸고 값도 0으로 바꾼다. 
	4. move 위치 인덱스에 해당 위치가 -1 일 경우 무조건 continue 한다.
	5. 바구니가 비어있으면 무조건 넣어준다.
	6. 바구니의 들어온 수와 top이 같으면 pop 을 하고 result + 2 를 해준다.
	7. 들어온 수와 top 이 다르면 들어온 수를 push 한다. 
	8. 이것을 move 만큼 반복한다. 
``` java
import java.util.*;
//1. board 맨 위의 위치와 수를 나타내는 클래스 배열을 N길이만큼 만든다. 
class InBox {
    static int LIMIT;
    int y;
    int num;
    
    InBox(int y, int num){
        if(y == LIMIT) this.y = -1;
        else this.y = y;
        updateNum(num);
    }
    
    InBox plusY(){
        this.y ++;
        if(y >= LIMIT) this.y = -1;
        return this;
    }
    
    InBox updateNum(int num){
        if(y == -1) this.num = -1;
        else this.num = num;
        return this;
    }
    
    @Override
    public String toString(){
        return "y: "+ y + ", num:" + num;
    }
}
class Solution {
    public int solution(int[][] board, int[] moves) {
        int result = 0;
        // 1. 배열 만들기
        int n = board.length;
        InBox.LIMIT = n;
        InBox[] topList = new InBox[n];
        Stack<Integer> bucket = new Stack<>();
        
        for(int x = 0; x < n; x++){
            int y = 0;
            while(true){
                int currentNum = board[y][x];
                if(currentNum != 0){
                    topList[x] = new InBox(y, currentNum);
                    break;
                }
                if(++y >= n) break;
            }
        }
        // System.out.println("topList" + Arrays.toString(topList));
            
        // 7. 이것을 move 만큼 반복한다. 
        int moveLen = moves.length;
        for(int i = 0; i < moveLen; i++){
            // 2. move 에서 현재 수가 있으면 해당 tops 의 위치로 가서 위치를 +1 한 다음 수를 가져와 바구니에 넣고 길이를 늘린다.
            int pickX = moves[i] - 1;
            InBox pickedTop = topList[pickX];
            if(pickedTop.y == -1) continue;
            // 바구니의 들어온 수와 top이 같으면 pop 을 하고 result + 2 를 해준다.
            int pickingNum = pickedTop.num;
            if(bucket.isEmpty()){
                bucket.push(pickingNum);
            } else {
                if(bucket.peek() == pickingNum){
                    bucket.pop();
                    result += 2;
                } else {
                    bucket.push(pickingNum);
                }
            }
            
            pickedTop.plusY();
            if(pickedTop.y == -1) {
                pickedTop.num = -1;
            }else{
            
                pickedTop.updateNum(board[pickedTop.y][pickX]);              
            }
        }
        return result;
    }
}
```
