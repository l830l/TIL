##### 몸 풀기 문제
###### 문제 25 트리 순회
- 문제
	- 저자권장 시간 40분 | 권장 시간 복잡도 O(N) | 출제 저자 출제
	- 이진 트리를 표현한 배열 `nodes`를 인자로 받습니다.
	- 예를 들어 `nodes`가 `[1, 2, 3, 4, 5, 6, 7]`이면 다음과 같은 트리를 표현한 것입니다.
		![[Pasted image 20260530230705.png]]
	- 해당 이진 트리에 대하여 전위 순회, 중위 순회, 후위 순회 결과를 반환하는 `solution()` 함수를 구현하세요
- 제약조건
	- 입력 노드값의 개수는 1개 이상 1,000개 이하이다.
	- 노드값은 정수형이며, 중복되지 않는다.
- 입출력의 예
	- \[1,2,3,4,5,6,7] > \["1245367", "4251637", "4526731"]
- 의사코드(배열로 풀어보기)
	1. 트리를 구축한다. 
		1. N 만큼 반복한다.
		2. 0번의 N\[0]는 arr\[1]로 둔다.
		3. N\[1]은 N\[0]의 왼쪽 자식이므로 2 * 1 = 2 즉, arr\[2] 에 둔다.
		4. N\[2]번은 N\[0]의 오른쪽 자식이므로 2 * 1 + 1 = 3. arr\[3]
		5. N\[3]번은 N\[1]의 왼쪽 자식이므로 arr\[2]의 왼쪽 자식이라서 2 * 2= 4. arr\[4]
		6. N\[4]번은 N\[1]의 오른쪽 자식이므로 2 * 2 + 1 = 5. arr\[5]
		7. N\[5]번은 3의 왼쪽 자식이므로 arr\[3]의 왼쪽 자식이라서 3 * 2  = 6. arr\[6]
		8. 즉, i 일 때 arr 의 인덱스는 i + 1
	2. 부모와 자식의 인덱스를 생각한다.
		1. 부모는 자식인덱스/2이고, 반대로 왼쪽 자식은 2N, 오른쪽 자식은 2N +1 임을 생각하며 푼다.
	3. 해당 인덱스를 바꿔서 전위 순회, 중위 순회, 후위 순회를 짠다. 방문한 것은 flag 를 1로 바꾸고 result 배열에 넣는다.
- 푼 코드 
``` java
import java.util.*;  
  
class Solution {  
    public String[] solution(int[] nums) {  
        // 트리를 구축한다.  
        int [] tree = new int[nums.length + 1];  
        int len = nums.length;  
        for(int i = 0; i < len; i++){  
            tree[i + 1]= nums[i];  
        }  
  
        return new String[]{  
                preOrder(tree, 1, len + 1), // 전위순회  
                inOrder(tree, 1, len + 1),  // 중위순회  
                postOrder(tree, 1, len + 1) // 후위순회  
        };  
    }  
  
    // 전위 순회  
    private String preOrder(int[] tree, int idx, int treeLen) {  
        StringBuilder sb = new StringBuilder();  
        sb.append(tree[idx]);  
        // 왼쪽 자식이 있으면 그쪽으로 이동하여 반복  
        if(idx * 2 < treeLen) {  
            sb.append(preOrder(tree, idx * 2, treeLen));  
        }  
        // 오른쪽 자식이 있으면 그쪽으로 이동하여 반복  
        if(idx * 2 + 1 < treeLen) {  
            sb.append(preOrder(tree, idx * 2 + 1, treeLen));  
        }  
        return sb.toString();  
    }  
  
    // 중위 순회  
    private String inOrder(int[] tree, int idx, int treeLen) {  
        StringBuilder sb = new StringBuilder();  
        // 왼쪽 자식이 있으면 그쪽으로 이동하여 반복  
        if(idx * 2 < treeLen) {  
            sb.append(inOrder(tree, idx * 2, treeLen));  
        }  
        sb.append(tree[idx]);  
        // 오른쪽 자식이 있으면 그쪽으로 이동하여 반복  
        if(idx * 2 + 1 < treeLen) {  
            sb.append(inOrder(tree, idx * 2 + 1, treeLen));  
        }  
        return sb.toString();  
    }  
      
    // 후위 순회  
    private String postOrder(int[] tree, int idx, int treeLen){  
        StringBuilder sb = new StringBuilder();  
        // 왼쪽 자식이 있으면 그쪽으로 이동하여 반복  
        if(idx * 2 < treeLen) {  
            sb.append(postOrder(tree, idx * 2, treeLen));  
        }  
        // 오른쪽 자식이 있으면 그쪽으로 이동하여 반복  
        if(idx * 2 + 1 < treeLen) {  
            sb.append(postOrder(tree, idx * 2 + 1, treeLen));  
        }  
        sb.append(tree[idx]);  
        return sb.toString();  
    }  
}
```
